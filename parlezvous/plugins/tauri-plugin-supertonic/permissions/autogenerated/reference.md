## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-ping`
- `allow-is-supertonic-ready`
- `allow-download-supertonic-models`
- `allow-generate-supertonic-tts`
- `allow-purge-supertonic-models`
