## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-ping`
- `allow-init-model`
- `allow-check-model-exists`
- `allow-download-model`
- `allow-purge-model`
- `allow-generate-chat`
- `allow-close-model`
- `allow-pick-gallery-image`
